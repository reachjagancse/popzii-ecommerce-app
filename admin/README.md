Popzii Admin — Panel

Overview
--------
React web app for admins to manage vendors, orders, and system analytics.

Features
--------
- Authentication
- Vendor approval/management
- Order monitoring
- System reports

Prerequisites
-------------
- Node.js 18+
- Backend running

Running Locally
---------------
1. Install:
   ```
   npm install
   ```
2. Start:
   ```
   npm start
   ```
3. Open http://localhost:3002

Validation
----------
- Login as admin
- View vendors
- Monitor orders
- Generate reports

API
---
- Base URL: http://localhost:4000
- Endpoints: /api/admin/*
